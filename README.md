# :airplane: BonVoyage
Projeto de Frontend, consiste na criação de uma página de agência de viagens, com 4 páginas :
- Home
- Pacotes e promoções
- Fale Conosco
- Cadastro

Contém também :
- Modal para Login
- Barra lateral de informações do perfil
